# Vertical Tab Memo

Chrome の縦タブを「メモの表示領域」として使うための、超軽量な静的 Web アプリ。

ページ自体の本文よりも、**Chrome の縦タブに並んだタイトル表示**を主役にした実験的なネタアプリです。  
URL パラメータで指定した文字列をページタイトルに反映し、複数タブを並べることで、縦タブそのものをメモ帳のように使えます。

---

## コンセプト

通常の Web アプリはページ本文が主役ですが、このアプリは違います。

- 主役はページ内UIではなく **ブラウザの縦タブ**
- 各ページは「1つのメモ」
- 複数タブを開くことで、縦に並ぶメモ一覧を作れる
- Chrome の正式リリースされた縦タブUIを、そのまま表示装置として遊ぶ

つまり、これはメモアプリというより、**Chrome のUIをハックして遊ぶミニマルな作品**です。

---

## 想定ユースケース

- Chrome の縦タブをメモ帳のように使う
- 1行メモを複数タブで並べる
- ネタ画像・動画を撮る
- SNS 投稿用に「本音メモ」「脳内会議」「人生のタブ」などを作る

例:

- 帰りたい
- 会議長い
- やること
- 明日やる
- 無理
- 理性
- 本能

---

## MVP

最初に作る最小機能は以下のみです。

### 必須機能

- URL クエリパラメータ `text` を受け取る
- `document.title` に反映する
- ページ本文にも同じテキストを表示する
- `text` 未指定時はデフォルト文言を表示する
- 文言は編集可能で、即座にクエリパラメータと`document.title` に反映する

### 例

```txt
/?text=帰りたい
/?text=会議長い
/?text=やること
````

## 技術方針

このアプリは極力シンプルに作ります。

* **静的サイト**
* 依存をほぼ持たない
* 必要になれば Vite 化する

初期段階では、複雑な状態管理やバックエンドは不要です。

---

## デザイン方針

* タブ側が主役なので、本文はミニマル
* 文字を大きめに表示してスクショ映えしやすくする
* 白背景またはかなりシンプルな背景
* 不要な UI は極力置かない
* 「Web アプリ感」より「実験作品感」を優先する

---

## SNS での見せ方

このアプリは使い勝手より、**見た瞬間に意味が伝わること**が重要です。

投稿時は以下のような見せ方を想定しています。

* Chrome の縦タブを表示した状態でスクショ
* 左側に複数のメモタイトルが並んでいる状態を見せる
* 本文よりも「縦タブがメモ帳化していること」が分かる構図にする

投稿例:

> Chrome の縦タブ、ページタイトルを並べるだけでメモ帳になるので、
> 任意のテキストを表示するだけのサイトを作った
